# Dev Tools
[![PyPI version](https://badge.fury.io/py/dev-tools.svg)](https://badge.fury.io/py/dev-tools)
## Usage
Run
```shell
dev_tools
```
## Installation
```shell
pip install dev-tools
```
or
```shell
pip install git+https://github.com/quintenroets/dev-tools.git
```
make sure to use Python 3.10+
